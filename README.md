⏳✨ Cosmic Focus - Cronómetro de Estudos Mágico

O Cosmic Focus é um cronómetro de estudos simples e interativo com um design inspirado no estilo espacial de Sailor Moon. Este projeto foi desenvolvido para praticar e consolidar os meus conhecimentos iniciais em desenvolvimento frontend.

🚀 Como Funciona?

Medidor de Tempo: Cronómetro preciso que exibe minutos, segundos e milésimos de segundo.

Fichário de Conquistas: Ao clicar em "Registrar", ganhas uma estrela (⭐) que é adicionada à tua lista de progresso com a data e hora exatas da conclusão do teu estudo.

Interface Responsiva: Feito para funcionar perfeitamente tanto no ecrã do computador como no telemóvel.

🛠️ Tecnologias Utilizadas

HTML5: Estrutura limpa da aplicação e do painel de conquistas.

CSS3 (com Tailwind CSS): Estilização moderna com efeito de vidro fosco (glassmorphism) e animações suaves de flutuação.

JavaScript: Lógica de tempo (através de setInterval), controlo dos botões de iniciar/pausar e criação dinâmica do histórico de estrelas.
